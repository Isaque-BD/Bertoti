Trade-Offs

#Windows X Linux
Aqui temos um exemplo bem comum de escolha de sistema operacional, vamos pegar o Windows, ele é simples e prático de usar para um usuário comum, podendo fazer de tudo. Porém o Windows não é muito otimizado exigindo um consumo a mais do hardware, e seu sistema de defesa não é muito bom, podendo deixar alguns vírus ser executados.

Agora Linux, é um sistema onde o usuário tem mais liberdade para mexer no source code, pois é open source, deixando várias possibilidades, seu sistema é muito bem otimizado, não exigindo tanto hardware, e seu sistema de defesa contra vírus é muito bom.Porém ele pode ser muito difícil para usuários sem experiência, sendo mais recomendando o Windows.

#Java x Python
O Java tem como grande vantagem sua portabilidade, e podendo ser programado em multiplataforma, porém é uma linguagem meio lenta. E uma linguagem bem abstrata para quem quer começar a aprender a programar.

Python é uma linguagem muito simples para programar e tanto para aprendizado, é bem legível e intuitivo, porém para desenvolvimento de software acaba sendo mais lento exigindo muito do hardware.

BD Relacional X NoSQL

BD Relacional ela é uma estrutura organizada, facilitando o acesso de dados, integridade de dados, escabilidade, porém ela é complexa na modelagem e admnistração de dados e tem uma dificuldade de escabailidade horizontal ao adicionar um novo servidor.

NoSQL é mais fléxivel ao aceitar vários tipos de dados, como imagens, vídeos, documentos e entre outros, oferece uma maior escabilidade e performance, porém sua modelagem é mais complexa que o SQL e menos suporte para transações, que é muito importante para a consistência de um banco de dados.
